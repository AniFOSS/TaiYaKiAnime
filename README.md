# TaiYaKi


Stream anime and sync your MyAnimeList (MAL) together with TaiYaKi!

  ### Features

  * Sync your list and watch anime at the same time, totally ad-free and no premium purchases necessary
  * Multiple sources to choose from
  * Uses the Jikan API 
  * Constantly updated!
